# Cravvy

Your food wingman. Chat naturally, get meal plans optimized for your budget and nutrition goals, compare prices across Zomato/Swiggy/Zepto, and discover what to eat next.

## Tech Stack

| Layer              | Tech                                      |
| ------------------ | ----------------------------------------- |
| Language           | Python 3.12+                              |
| API                | FastAPI + uvicorn                          |
| Package Manager    | uv                                        |
| Database           | MongoDB (Motor async driver)              |
| Cache / Sessions   | Redis                                     |
| AI                 | Groq API (free tier + BYO key support)     |
| MCP Layer          | FastMCP v3 (server composition)            |
| Optimization       | Google OR-Tools CP-SAT                     |
| Personalization    | Thompson Sampling (Bayesian)               |
| Email              | Resend                                     |
| Lint / Format      | ruff                                       |
| Type Check         | pyright (strict)                           |
| Tests              | pytest + pytest-asyncio                    |
| Deployment         | Azure                                      |

## Architecture

```
src/
  api/            → FastAPI app, middleware, route registration
  core/           → Shared models
  components/     → Built-in implementations (LLM, MCP, optimization)
  auth/           → Signup, login, JWT, OAuth, password reset
  users/          → User profiles, preferences, dietary info
  chat/           → AI chat orchestration, streaming
  meals/          → Meal planning, OR-Tools optimization, history
  integrations/   → Zomato, Swiggy, Zepto OAuth + MCP connections
  shared/         → Config, database, utilities
tests/
  unit/           → Mocked services
  integration/    → Real services (requires docker compose)
  eval/           → Accuracy and quality evaluation
scripts/          → Utility scripts
```

## Pricing Model

- **Free tier**: 25 messages (Groq-powered)
- **Subscribed (₹49)**: 60 messages per day
- **BYO Key**: Paste your own provider + API key, unlimited usage

## Quick Start

```bash
# 1. Clone
git clone <repo-url>
cd cravvy

# 2. Install dependencies
uv sync

# 3. Start local services
docker compose up -d

# 4. Configure environment
cp .env.example .env
# Fill in your API keys

# 5. Run server
uv run python main.py
```

Server starts at `http://localhost:8000`. Health check at `GET /health`.

## Key Concepts

- **Domain Modules**: Each feature (auth, users, chat, meals) is self-contained with its own router, service, schemas, models, and exceptions.
- **Service Layer**: Business logic lives in services, not routes. Routes are thin — validate input, call service, return response.
- **MCP Composition**: Multiple MCP servers (Zomato, Swiggy, Zepto, nutrition) mounted under one FastMCP gateway with namespaced tools.
- **OR-Tools Optimization**: Weekly meal plans generated by constraint satisfaction solver, not LLM guessing.
- **Thompson Sampling**: Recommendations improve with every thumbs up/down — no ML pipeline needed.
